### Skills to Know About
- `/clear` — Clears the conversation context and starts fresh. Used very frequently here (57x/month) — reach for it whenever a task is complete and you're starting something new, to keep Claude's context focused and avoid confusion from prior sessions.
- `/plan` — Puts Claude into planning mode before it writes any code. Use it when starting a non-trivial feature or architectural change so you can review the approach before Claude commits to an implementation.
- `/model` — Switches the active Claude model mid-session. Useful for switching to Opus when you need deeper reasoning on a hard architecture problem, or to Haiku for quick lookups.
- `/exit` — Exits the Claude Code session cleanly.
- `/batch` — Runs a prompt against multiple files or inputs at once. Handy for bulk operations like checking all agent files for a consistent pattern.
